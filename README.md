# rpg
RPG Python Text based adventure for TLG course

Winning Condition: 
> Defeat all the Pokemon in the game.

Locations of Pokemon:
> Pantry: Charizard
> Garden: Bellsprout
> Kitchen: Meowth

## Features that were added to the original game: 
  > **Combat System**
  > **Loot System**
  > **New Winning Condition**
